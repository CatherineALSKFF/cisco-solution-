# Dummy Contract Corpus

This folder contains 20 synthetic software and commercial contracts for hackathon testing.

What is included:
- 20 fictional contract documents in Markdown
- `contract_index.csv` for bulk loading into a simple database or spreadsheet
- `contract_index.json` for demos, embeddings, or prompt pipelines

Design principles:
- Based on the agreement families visible in the attached Cisco-related contract filenames
- No actual contractual text reproduced
- Realistic fields for AI interrogation, including signature state, last review, auto-renewal, commercial model, and replacement notes

Suggested demo questions:
- Which agreements are still on vendor paper?
- Which contracts have not been reviewed in more than three years?
- Which active agreements auto-renew and also have royalty reporting obligations?
- Which contracts look replaceable by open source or internal development?
- Which agreements appear high-risk because they are old, still active, and misaligned with standard terms?
